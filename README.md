# Interactive 3D Experience – README & Walkthrough

## 1. Project Title
*ITD X DDA ASG1 – PetPals AR*

---

## 2. Description of the Game / Application
This Unity project is a first-person interactive experience featuring AR Virtual Pets  
Players scan AR pet posters, interact with a pet system (petting and feeding), and do actions to evolve their pet.

Main features include:
- AR poster scanning to spawn pets
- Pet system (happiness/XP level, hunger, levelling up)
- Toys interaction system

---

## 3. Installation & Setup Instructions

### 3.1 Requirements
| Component | Requirement |
|----------|-------------|
| Platform | Android Phone / iPhone |
| Hardware | A smartphone capable of running AR applications |
| Controls | Touchscreen |
| Engine | Unity 2022/2023 LTS |

### 3.2 How to Run the Game
1. Download the *APK* (Android) or *iOS TestFlight build* .  
2. Install and launch the app.  
3. Allow camera permissions.  
4. Point your camera at a pet AR poster to spawn your pet.

## 6. Pet System

### 6.1 Happiness
- Petting increases happiness by +10.  
- Toys increase happiness based on toy type.
- When happiness reaches the required amount, the pet levels up.

### 6.2 Hunger
- Every pet has a hunger bar
- Hunger bar decreases by *1 bar per minute*.  
- Feeding restores *1 bar*.  
- If hunger reaches 0, the pet dies.

---

## 7. Toy System

### How Toys Work
1. Open the *Toys Owned* page.  
2. Press *Use Now* on any toy to spawn the toy model.  
3. Drag the toy around the scene.  
4. When the toy touches the pet, the toy disappears.  
5. The pet gains happiness.  
6. The toy enters a cooldown (e.g., 3 hours).
